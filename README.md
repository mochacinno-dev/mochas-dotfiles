# Mocha's Dotfiles!
Hello there! These are my `.config` files for things I usually use. Most of these are taken from other repos and slightly modified to my liking!. I also have a few wallpapers.

Most of these are based on Catppuccin... I just really love Catppuccin.

### Apps Supported:
- Kitty
- Fastfetch
- Hyprland
- Rofi
- Waybar
- WLogout
